# 把简历 HTML 导出成能投递的 PDF

offer-helper 生成的是一份 A4 排版好的 `.html` 简历。投递平台大多要 PDF，
按下面三步导出（整个过程 1 分钟）。

## 通用三步（Chrome / Edge）

1. **打开 HTML**：双击生成的 `resume-xxx.html` 文件，用浏览器打开。
   - Claude Code 用户：文件在 `~/offer-helper-data/` 下。
   - 网页版用户：把 Claude 给你的 HTML 文本存成一个 `简历.html` 文件再打开
     （新建记事本/文本文件，把代码粘进去，文件名后缀改成 `.html`）。
2. **打印**：按 `Ctrl/Cmd + P` 调出打印窗口。
3. **存为 PDF**，并核对这几个设置：
   - 目标 / 打印机：选「**另存为 PDF**」
   - 纸张大小：**A4**
   - 边距：**默认**（模板已留白，无需再加）
   - **勾选「背景图形 / Background graphics」**（否则标题色块、分割线会丢）
   - 关闭「页眉和页脚」（去掉网址和页码）

点保存，得到一份干净的 A4 PDF。

## 常见问题

- **简历溢出到第 2 页只有一两行**：让 offer-helper「精简到单页」，它会先砍
  低权重 bullet；或自己删掉最不相关的一条经历。
- **中文变成方块**：换一台装了中文字体的设备打印，或让 offer-helper 把字体
  改成你系统里有的中文字体。
- **色块/分割线没了**：你忘了勾「背景图形」，回到第 3 步勾上。
- **手机上没有打印另存 PDF**：建议用电脑导出；临时方案是用支持「网页转 PDF」
  的 App 打开该 HTML。
