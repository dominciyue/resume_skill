# offer-helper 宣传视频 · 制作方案 + 可直接照拍脚本

目标：60–90 秒短视频，发抖音 / B站 / 小红书，让人秒懂"这个 skill 帮我写简历
不编数字，还能当面试官把我问穿"，并去 GitHub 安装。

## 一、核心打法（先定调子）

- **一句话钩子**：「让 AI 帮你改简历，它会顺着你吹；这个 skill 反过来——先把你
  问到答不上来。」反差感是传播点。
- **不要做成功能罗列**，要做成**一个真实的小故事**：水简历 → 被改 → 被面试官
  问穿 → 补牢 → 投出去。情绪曲线：心虚 → 爽 → 服气。
- **真实 > 精致**：用你自己仓库跑出来的真实对话录屏，比任何动画都有说服力。

## 三、工具与具体操作（小白友好）

**录屏（拍真实素材）**
- 终端录屏：用 OBS Studio（免费）录 Claude Code 跑 demo 的全过程；或用
  asciinema 录终端再转 gif/视频。
- 关键：录之前先按 `examples/demo.md` 真跑一遍，确认对话精彩（尤其"被问穿"
  那一下），再正式录。字号调大，方便手机上看清。

**剪辑 + 动画 + 转场 + 配音：用剪映（CapCut）**——一个软件全搞定，最适合小白：
- **转场**：在两段素材中间点"转场"，选「叠化/推拉/放大」即可（对应脚本里的
  滑入、上滑）。别用太花哨的，统一 1–2 种最干净。
- **动画**：选中文字/贴纸 → "动画" → 入场选「打字机/弹入」、强调选「放大」。
  脚本里的红✗绿✓、放大镜 zoom，用剪映"贴纸"+"蒙版放大"实现。
- **独白（配音）**：两条路——① 自己用手机录旁白导入；② 剪映"文本朗读"用 AI
  音色（推荐"阳光男声/知性女声"），把下面独白文案贴进去一键生成，省事且清晰。
- **字幕**：剪映"识别字幕"自动把配音转成字幕，再把关键句放大加粗描边。
- **BGM**：选轻快、节奏感强的纯音乐，音量压到 -18dB 别盖住人声；"被问穿"处
  加一个停顿/重音。
- **节奏**：每 3–5 秒必须有画面变化（切镜/动画/zoom），否则划走。开头 3 秒
  必须是钩子，决定完播率。

**封面**：大字「AI 帮我吹的简历，被它当场拆穿」+ 面试对话截图，制造好奇。

## 四、不同平台的小调整

- **抖音/快手**：竖屏 9:16，节奏更快，钩子前置到 0–2 秒，多用音效。
- **B站**：可放横屏、稍长（2–3 分钟），加更多真实对话过程，标题党适度。
- **小红书**：竖屏 + 图文笔记并发，封面写"求职 | 让 AI 当面试官捶我"，正文带
  安装步骤截图。

## 五、独白全文（可直接贴进剪映文本朗读）

> 你有没有发现，让 AI 改简历，它总爱帮你把数字往大了吹？可吹出来的，面试一问
> 就穿。这个 skill 反着来。把简历、GitHub 仓库丢给它，自动整理成你的经历库，

> 一直问到你卡住为止。最后告诉你每个点的真实天花板，面试前该补哪一层，一清二楚。
> 免费、开源，一行命令装好，链接在评论区。
