# offer-helper（上岸 Offer 助手）设计文档

日期：2026-06-05
状态：已批准（方向与 §1 流程经用户确认；§2-§5 按「先做、中途讨论」模式推进）

## 1. 背景与定位

基于市场调研（2026-06）：
- Agent Skills 生态中非编程类 skill 供给严重不足，中文求职场景 skill 完全空白；73% 社区 skill 败在质量。
- 需求侧：中国每年 1200 万+ 毕业生；70%+ 求职者因面试准备不足失败；大三大四学生高频使用 AI 做"求职建议、模拟面试"（中青报×Soul 2025）。
- 竞品全部是收费 App（鹅来面、JobLeap、智联模拟面试间），skill 形态的差异化：免费、数据全本地、跨运行时（Claude Code / Claude.ai / Cursor / Codex）、经历库一次填写多 JD 复用。

**一句话定位**：装进 Claude 里的免费求职教练——经历库只写一次，每个 JD 秒级定制简历，再用反向追问式压力面试把简历"焊牢"。

**目标用户**：会用 AI 工具的大学生与 0-3 年职场人（秋招/春招/实习/跳槽）。

## 2. MVP 范围（已确认：双模块）

- ✅ 经历库（中枢数据）
- ✅ 简历优化模块（JD 分析 → 经历匹配 → STAR 量化改写 → HTML/PDF 输出）
- ✅ 压力模拟面试模块（反向追问 + 漏洞清单 + 反哺经历库）
- ❌ 不做：行业面经题库（v2）、求职进度管理、英文简历（v2）、外部 API 依赖

## 3. Skill 结构

```
offer-helper/
├─ SKILL.md                    # 入口：触发条件、流程路由（建库/简历/面试）
├─ resume/
│   ├─ jd-analysis.md          # JD 关键词与能力要求提取方法
│   ├─ star-rules.md           # STAR 量化改写规则与正反例
│   └─ anti-fabrication.md     # 防虚构红线：所有内容可溯源到经历库
├─ interview/
│   ├─ pressure.md             # 反向追问式压力面流程与追问策略
│   └─ feedback.md             # 逐题点评 + 简历漏洞清单输出格式
├─ templates/
│   └─ resume.html             # 中文 A4 简历模板（打印即 PDF）
└─ experience/
    └─ profile.template.md     # 经历库模板（用户首次访谈后生成 profile.md）
```

设计原则（来自调研教训）：SKILL.md 保持精简（<300 行），细则下沉到子文件按需加载；每个子文件单一职责。

## 4. 核心流程

### 4.1 首次建库（约 15 分钟）
引导式访谈逐段挖掘：教育 / 项目 / 实习 / 技能。每段经历必问：背景、本人具体动作、可量化结果、踩过的坑。产出 `experience/profile.md`（用户本地）。

### 4.2 定制简历（每次 1-2 分钟）
输入 JD → 提取关键词/能力项 → 从经历库选最匹配经历 → STAR 量化改写（防虚构校验）→ 输出简历 HTML + 匹配度报告（命中哪些要求、缺口在哪）。

### 4.3 压力模拟面试
基于当前简历 + JD，面试官角色连环追问（角色、KPI 口径、最难点、备选方案）。结束输出：逐题点评、简历漏洞清单、经历库回改建议。

## 5. 错误处理与质量保障

- **防虚构**：改写内容必须溯源经历库原文；缺数据时引导用户补充而非编造；红线写入 anti-fabrication.md 并在简历输出前自检。
- **经历库缺失**：任一流程检测不到 profile.md 时引导先建库。
- **JD 质量差**（过短/含糊）：提示用户补充岗位信息而非硬猜。
- **测试**：用 3-5 套真实风格的「经历 × JD」组合做端到端验收（应届技术岗、应届非技术岗、社招跳槽）；检查防虚构（故意留缺口的经历库不得被编造填充）；模板在浏览器打印 A4 不破版。

## 6. 发布策略

1. GitHub 开源仓库（中文 README + 动图演示），按 Agent Skills 开放标准组织，标注兼容 Claude Code / Claude.ai / Cursor / Codex。
2. 提交收录：claude-code-skills-zh、awesome-claude-skills 系列合集、SkillsMP 等市场。
3. 传播场景：求职社群、牛客、小红书（晒简历 before/after 对比）。
4. v2 路线：行业面经题库 → 英文简历 → 考公搭子 skill（复用经历库结构）。
